# SQuAD2-style dataset

This dataset is generated from the data on the biodegradability of polyesters that was extracted from [1] algorithmically using pre-defined templates. Each entry contains "context", "question", "answers" (answer is given with the location of the start symbol, it should be exactly locatable in context). There are about 20% of unanswerable questions. In this dataset, each context contains information about one polymer. 

## References
